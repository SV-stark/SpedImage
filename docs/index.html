Built for Speed and Safety

Rebuilt from the ground up in Rust + WGPU. Zero CPU overhead for real-time manipulation adjustments.

⚡

Zero-Copy Loading

Decodes directly to memory and loads onto the GPU as a mapped texture. Minimizes CPU bottleneck and cuts startup rendering delay below 100ms.

🎮

GPU-Accelerated Shaders

All photo edits (brightness, contrast, saturation, and crop zoom panning) are compiled as WGSL shaders. Runs instantly on 4K/8K images without lagging the main thread.

🦀

Rust Safe Core

Utilizes Rust's ownership model to guarantee absolute safety, eliminate memory leaks, and run multi-threaded prefetching safely across your folders.

🔄

Rayon Parallel Rotation

Automated auto-orientation (via EXIF metadata) uses multi-core CPU parallel block resizing via Rayon to guarantee no rendering hitches during folder slides.

📸

Format agnostic

Built-in support for RAW camera formats, HEIF/AVIF, JXL, SVG, animated GIFs, and standard image files (JPEG, PNG, TIFF, WebP).

⚡

Native Productivity Gestures

Double-click to toggle fullscreen, press Enter to view at pixel-perfect 1:1 scale, copy file path with Ctrl+Shift+C, and safely delete to Recycle Bin instantly.

🗺️

Color Space & GPS Lookup

Detects Adobe RGB color tags to run precise gamut transforms in shaders. Plus, single-click GPS mapping coordinates directly to web maps.

🎛️

Window State Persistence

Remembers your window bounds, sidebar view preferences, and thumbnail strip settings across launches for a seamless interface experience.

Get SpedImage

Install SpedImage on your machine and start browsing your images instantly.

Install via Winget (Windows Package Manager)

winget install SV-stark.SpedImage

Install via Cargo (Rust package manager)

cargo install --git https://github.com/SV-stark/SpedImage

Clone and compile from source

git clone https://github.com/SV-stark/SpedImage.git && cd SpedImage && cargo build --release
